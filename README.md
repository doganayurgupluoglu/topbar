# Topbar with Search

Kullanışlı bir navigation bar tasarımı. Arama çubuğu, giriş ve kayıt olma butonları içerir.

## Özellikler

- Responsive tasarım
- Entegre arama çubuğu
- Hover efektleri
- Giriş ve kayıt olma butonları
- Font Awesome ikonları

## Kullanılan Teknolojiler

- HTML5
- CSS3
- Font Awesome
